Cancer du sein

Description

Le cancer du sein est une maladie où des cellules malignes (cancéreuses) se forment dans les tissus du sein, souvent dans les canaux (conduits qui transportent le lait) ou les lobules (glandes productrices de lait). Cette maladie peut se propager (métastases) aux tissus environnants ou à d'autres parties du corps. Les trois formes les plus fréquemment rencontrées sont : les carcinomes non infiltrant, les carcinomes infiltrants, les carcinomes inflammatoires.Des avancées récentes en recherche permettent d'améliorer constamment les traitements disponibles.
